# Excel-Sales-Dashboard

## Project Overview
Interactive Excel dashboard analyzing sales & profit by region, product, and month.
The dashboard uses Pivot Tables, Charts, and Slicers to provide actionable insights.
## Key Features
- **Interactive Pivot Tables** for total sales, total profit, and monthly trends.
- **Charts**: Column chart for sales by region, Pie chart for profit contribution by product, Line chart for monthly sales trend.
- **Slicers**: Filter data dynamically by `Region` and `Product`.
- **KPIs**: Quick view of Total Sales and Total Profit.

## Skills Applied
- Excel Formulas & Functions
- Pivot Tables & Charts
- Slicers and Interactive Dashboards
- Data Visualization
